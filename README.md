# Road Trip Spending Calculator

Useful tool in road trips to share the expenses in a large groups of people.

* Backbone.JS
* jQuery Mobile
* i18n friendly
* qunit

## Quick Start

```bash
npm install
npm start
```
open: http://localhost:8080/
